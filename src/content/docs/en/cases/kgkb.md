## 技术栈
- **后端：** FastAPI + SQLite + ChromaDB
- **前端：** React + TypeScript + D3.js + Tailwind CSS
- **开发模式：** 夜间工厂 —— 人睡觉，Agent 写代码

## 工厂流水线
```
任务定义 → Code Agent 执行 → 自动测试 → Git 提交 → 进度汇报
```

## 成果
- **30 个任务** 在 **3 天** 内完成
- 平均 **8-15 分钟/任务**
- **5000+ 行** 代码
- **< 10%** 人工介入
