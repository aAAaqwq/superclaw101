## ⚡ 快速开始

```bash
# 安装 OpenClaw
npm install -g openclaw

# 初始化你的第一个 Agent
openclaw init

# 连接消息渠道（Telegram / 飞书 / Discord...）
openclaw channels add

# 启动网关
openclaw gateway
```

就这么简单，你的 AI Agent 已上线。[完整配置指南 →](/getting-started/)
